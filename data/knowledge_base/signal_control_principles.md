# Signal Control Principles

This is a sample traffic knowledge base document for demos and interviews.

## 信控优化注意事项
信控优化建议只能用于辅助分析，不能直接下发真实信号控制策略。任何候选方案都需要结合排队长度、饱和度、相位安全、行人过街需求、仿真验证和人工审核。

## 信号配时安全边界
Agent 可以解释周期、绿信比、相位和协调控制原则，但不能替代交管部门决策，也不能声称已经控制红绿灯。

## Optimization Objectives
Signal optimization aims to reduce delay, queue length and spillback while maintaining pedestrian and side-street safety.

## Cycle Length and Green Ratio
Cycle length and green ratio should be evaluated with demand, saturation and queue observations.

## Coordination Control
Adjacent intersections can be coordinated, but proposed plans require simulation and manual review.

## Safety Boundary
This Agent must not directly issue real signal control strategies. Any recommendation is for analysis only and requires simulation validation and human approval.

## Manual Review
Human traffic engineers must review candidate timing changes before deployment.
